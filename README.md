# Mobile_Sales_Dashboard

## 📌 Project Overview
The **Mobile Sales Dashboard** is an interactive dashboard developed using Power BI to analyze and visualize mobile sales performance. This project helps in understanding important business metrics such as total sales, revenue, profit, units sold, and product performance through attractive and easy-to-understand visualizations.

The dashboard includes multiple charts and interactive elements to provide meaningful insights and support better decision-making. It is designed with a professional layout, attractive colors, and user-friendly navigation for effective sales analysis.

---

## 🚀 Features
- 📊 Interactive Mobile Sales Dashboard
- 📈 Bar Graph for Sales Analysis
- 🥧 Pie Chart for Product Distribution
- 🎯 Funnel Chart for Sales Performance
- 💳 KPI Cards for Key Metrics
- 📉 Income/Sales Trend Chart
- 📋 Data Table for Detailed Information
- 🔘 Interactive Mobile Selection Buttons
- 🎨 Professional and Attractive Dashboard Design
